# odin-bubble-sort
A [bubble sort](https://www.theodinproject.com/lessons/ruby-bubble-sort) implementation, written in Ruby, for [The Odin Project](https://www.theodinproject.com/).

As my fourth project in the Ruby track, this project has taught or reinforced for me:
- I still remember at least a few things from my decade-old C++ coursework
- Even the most basic of loops still have their uses at times
- Ruby does not have a for(;;) loop, making each_with_index is my new favorite enumeration
- Conditionals at the end of statements flow so nicely off the tongue
